# 大学生就业信息管理系统数据库设计

## 1. 数据库表结构

### 1.1 用户表（user）

| 字段名      | 数据类型                            | 约束                                                            | 描述             |
| ----------- | ----------------------------------- | --------------------------------------------------------------- | ---------------- |
| id          | BIGINT                              | PRIMARY KEY, AUTO_INCREMENT                                     | 用户ID           |
| username    | VARCHAR(50)                         | UNIQUE, NOT NULL                                                | 用户名           |
| password    | VARCHAR(100)                        | NOT NULL                                                        | 密码（加密存储） |
| role        | ENUM('student', 'company', 'admin') | NOT NULL                                                        | 角色             |
| status      | ENUM('active', 'inactive')          | NOT NULL, DEFAULT 'active'                                      | 状态             |
| create_time | DATETIME                            | NOT NULL, DEFAULT CURRENT_TIMESTAMP                             | 创建时间         |
| update_time | DATETIME                            | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间         |

### 1.2 学生表（student）

| 字段名      | 数据类型               | 约束                                                            | 描述       |
| ----------- | ---------------------- | --------------------------------------------------------------- | ---------- |
| id          | BIGINT                 | PRIMARY KEY, AUTO_INCREMENT                                     | 学生ID     |
| user_id     | BIGINT                 | NOT NULL, FOREIGN KEY REFERENCES user(id)                       | 关联用户ID |
| name        | VARCHAR(50)            | NOT NULL                                                        | 姓名       |
| gender      | ENUM('male', 'female') | NOT NULL                                                        | 性别       |
| student_id  | VARCHAR(20)            | UNIQUE, NOT NULL                                                | 学号       |
| major       | VARCHAR(100)           | NOT NULL                                                        | 专业       |
| grade       | VARCHAR(20)            | NOT NULL                                                        | 年级       |
| phone       | VARCHAR(20)            | NOT NULL                                                        | 手机号     |
| email       | VARCHAR(100)           | NOT NULL                                                        | 邮箱       |
| avatar      | VARCHAR(255)           |                                                                 | 头像路径   |
| create_time | DATETIME               | NOT NULL, DEFAULT CURRENT_TIMESTAMP                             | 创建时间   |
| update_time | DATETIME               | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间   |

### 1.3 企业表（company）

| 字段名        | 数据类型                                | 约束                                                            | 描述         |
| ------------- | --------------------------------------- | --------------------------------------------------------------- | ------------ |
| id            | BIGINT                                  | PRIMARY KEY, AUTO_INCREMENT                                     | 企业ID       |
| user_id       | BIGINT                                  | NOT NULL, FOREIGN KEY REFERENCES user(id)                       | 关联用户ID   |
| company_name  | VARCHAR(100)                            | NOT NULL                                                        | 企业名称     |
| industry      | VARCHAR(100)                            | NOT NULL                                                        | 行业         |
| scale         | VARCHAR(50)                             | NOT NULL                                                        | 规模         |
| address       | VARCHAR(255)                            | NOT NULL                                                        | 地址         |
| contact_name  | VARCHAR(50)                             | NOT NULL                                                        | 联系人       |
| contact_phone | VARCHAR(20)                             | NOT NULL                                                        | 联系电话     |
| contact_email | VARCHAR(100)                            | NOT NULL                                                        | 联系邮箱     |
| logo          | VARCHAR(255)                            |                                                                 | 企业logo路径 |
| status        | ENUM('pending', 'approved', 'rejected') | NOT NULL, DEFAULT 'pending'                                     | 审核状态     |
| create_time   | DATETIME                                | NOT NULL, DEFAULT CURRENT_TIMESTAMP                             | 创建时间     |
| update_time   | DATETIME                                | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间     |

### 1.4 岗位表（job）

| 字段名       | 数据类型                  | 约束                                                            | 描述       |
| ------------ | ------------------------- | --------------------------------------------------------------- | ---------- |
| id           | BIGINT                    | PRIMARY KEY, AUTO_INCREMENT                                     | 岗位ID     |
| company_id   | BIGINT                    | NOT NULL, FOREIGN KEY REFERENCES company(id)                    | 关联企业ID |
| job_title    | VARCHAR(100)              | NOT NULL                                                        | 职位名称   |
| salary_min   | DECIMAL(10,2)             | NOT NULL                                                        | 最低薪资   |
| salary_max   | DECIMAL(10,2)             | NOT NULL                                                        | 最高薪资   |
| location     | VARCHAR(100)              | NOT NULL                                                        | 工作地点   |
| education    | VARCHAR(50)               | NOT NULL                                                        | 学历要求   |
| experience   | VARCHAR(50)               | NOT NULL                                                        | 经验要求   |
| industry     | VARCHAR(100)              | NOT NULL                                                        | 行业       |
| description  | TEXT                      | NOT NULL                                                        | 职位描述   |
| requirements | TEXT                      | NOT NULL                                                        | 职位要求   |
| status       | ENUM('active', 'expired') | NOT NULL, DEFAULT 'active'                                      | 状态       |
| publish_time | DATETIME                  | NOT NULL, DEFAULT CURRENT_TIMESTAMP                             | 发布时间   |
| update_time  | DATETIME                  | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间   |

### 1.5 简历表（resume）

| 字段名      | 数据类型                   | 约束                                                            | 描述       |
| ----------- | -------------------------- | --------------------------------------------------------------- | ---------- |
| id          | BIGINT                     | PRIMARY KEY, AUTO_INCREMENT                                     | 简历ID     |
| student_id  | BIGINT                     | NOT NULL, FOREIGN KEY REFERENCES student(id)                    | 关联学生ID |
| resume_name | VARCHAR(100)               | NOT NULL                                                        | 简历名称   |
| basic_info  | JSON                       | NOT NULL                                                        | 基本信息   |
| education   | JSON                       | NOT NULL                                                        | 教育背景   |
| experience  | JSON                       |                                                                 | 工作经历   |
| projects    | JSON                       |                                                                 | 项目经验   |
| skills      | JSON                       |                                                                 | 技能证书   |
| status      | ENUM('draft', 'published') | NOT NULL, DEFAULT 'draft'                                       | 状态       |
| create_time | DATETIME                   | NOT NULL, DEFAULT CURRENT_TIMESTAMP                             | 创建时间   |
| update_time | DATETIME                   | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间   |

### 1.6 投递记录表（application）

| 字段名      | 数据类型                                                        | 约束                                                            | 描述       |
| ----------- | --------------------------------------------------------------- | --------------------------------------------------------------- | ---------- |
| id          | BIGINT                                                          | PRIMARY KEY, AUTO_INCREMENT                                     | 投递记录ID |
| student_id  | BIGINT                                                          | NOT NULL, FOREIGN KEY REFERENCES student(id)                    | 关联学生ID |
| job_id      | BIGINT                                                          | NOT NULL, FOREIGN KEY REFERENCES job(id)                        | 关联岗位ID |
| resume_id   | BIGINT                                                          | NOT NULL, FOREIGN KEY REFERENCES resume(id)                     | 关联简历ID |
| status      | ENUM('pending', 'reviewed', 'interview', 'offered', 'rejected') | NOT NULL, DEFAULT 'pending'                                     | 状态       |
| create_time | DATETIME                                                        | NOT NULL, DEFAULT CURRENT_TIMESTAMP                             | 投递时间   |
| update_time | DATETIME                                                        | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间   |

### 1.7 评价表（evaluation）

| 字段名       | 数据类型                   | 约束                                    | 描述       |
| ------------ | -------------------------- | --------------------------------------- | ---------- |
| id           | BIGINT                     | PRIMARY KEY, AUTO_INCREMENT             | 评价ID     |
| evaluator_id | BIGINT                     | NOT NULL                                | 评价人ID   |
| evaluatee_id | BIGINT                     | NOT NULL                                | 被评价人ID |
| type         | ENUM('company', 'student') | NOT NULL                                | 评价类型   |
| score        | INT                        | NOT NULL, CHECK (score BETWEEN 1 AND 5) | 评分       |
| content      | TEXT                       | NOT NULL                                | 评价内容   |
| create_time  | DATETIME                   | NOT NULL, DEFAULT CURRENT_TIMESTAMP     | 创建时间   |

### 1.8 岗位收藏表（job_collection）

| 字段名      | 数据类型             | 约束                                         | 描述                   |
| ----------- | -------------------- | -------------------------------------------- | ---------------------- |
| id          | BIGINT               | PRIMARY KEY, AUTO_INCREMENT                  | 收藏ID                 |
| student_id  | BIGINT               | NOT NULL, FOREIGN KEY REFERENCES student(id) | 关联学生ID             |
| job_id      | BIGINT               | NOT NULL, FOREIGN KEY REFERENCES job(id)     | 关联岗位ID             |
| create_time | DATETIME             | NOT NULL, DEFAULT CURRENT_TIMESTAMP          | 收藏时间               |
| UNIQUE KEY  | (student_id, job_id) |                                              | 唯一约束，防止重复收藏 |

### 1.9 面试记录表（interview）

| 字段名             | 数据类型                                    | 约束                                                            | 描述           |
| ------------------ | ------------------------------------------- | --------------------------------------------------------------- | -------------- |
| id                 | BIGINT                                      | PRIMARY KEY, AUTO_INCREMENT                                     | 面试记录ID     |
| application_id     | BIGINT                                      | NOT NULL, FOREIGN KEY REFERENCES application(id)                | 关联投递记录ID |
| interview_time     | DATETIME                                    | NOT NULL                                                        | 面试时间       |
| interview_location | VARCHAR(255)                                | NOT NULL                                                        | 面试地点       |
| interview_type     | VARCHAR(50)                                 | NOT NULL                                                        | 面试类型       |
| interviewer        | VARCHAR(50)                                 | NOT NULL                                                        | 面试官         |
| notes              | TEXT                                        |                                                                 | 备注           |
| status             | ENUM('scheduled', 'completed', 'cancelled') | NOT NULL, DEFAULT 'scheduled'                                   | 状态           |
| create_time        | DATETIME                                    | NOT NULL, DEFAULT CURRENT_TIMESTAMP                             | 创建时间       |
| update_time        | DATETIME                                    | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间       |

### 1.10 系统日志表（system_log）

| 字段名      | 数据类型     | 约束                                | 描述       |
| ----------- | ------------ | ----------------------------------- | ---------- |
| id          | BIGINT       | PRIMARY KEY, AUTO_INCREMENT         | 日志ID     |
| user_id     | BIGINT       |                                     | 操作用户ID |
| operation   | VARCHAR(100) | NOT NULL                            | 操作类型   |
| description | TEXT         | NOT NULL                            | 操作描述   |
| ip_address  | VARCHAR(50)  |                                     | IP地址     |
| create_time | DATETIME     | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间   |

## 4. 数据库初始化脚本

### 4.1 创建数据库

```sql
CREATE DATABASE IF NOT EXISTS employment_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

USE employment_system;
```

### 4.2 创建用户表

```sql
CREATE TABLE IF NOT EXISTS user (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(100) NOT NULL,
    role ENUM('student', 'company', 'admin') NOT NULL,
    status ENUM('active', 'inactive') NOT NULL DEFAULT 'active',
    create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 4.3 创建学生表

```sql
CREATE TABLE IF NOT EXISTS student (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    name VARCHAR(50) NOT NULL,
    gender ENUM('male', 'female') NOT NULL,
    student_id VARCHAR(20) UNIQUE NOT NULL,
    major VARCHAR(100) NOT NULL,
    grade VARCHAR(20) NOT NULL,
    phone VARCHAR(20) NOT NULL,
    email VARCHAR(100) NOT NULL,
    avatar VARCHAR(255),
    create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES user(id) ON DELETE CASCADE
);
```

### 4.4 创建企业表

```sql
CREATE TABLE IF NOT EXISTS company (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    company_name VARCHAR(100) NOT NULL,
    industry VARCHAR(100) NOT NULL,
    scale VARCHAR(50) NOT NULL,
    address VARCHAR(255) NOT NULL,
    contact_name VARCHAR(50) NOT NULL,
    contact_phone VARCHAR(20) NOT NULL,
    contact_email VARCHAR(100) NOT NULL,
    logo VARCHAR(255),
    status ENUM('pending', 'approved', 'rejected') NOT NULL DEFAULT 'pending',
    create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES user(id) ON DELETE CASCADE
);
```

### 4.5 创建岗位表

```sql
CREATE TABLE IF NOT EXISTS job (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    company_id BIGINT NOT NULL,
    job_title VARCHAR(100) NOT NULL,
    salary_min DECIMAL(10,2) NOT NULL,
    salary_max DECIMAL(10,2) NOT NULL,
    location VARCHAR(100) NOT NULL,
    education VARCHAR(50) NOT NULL,
    experience VARCHAR(50) NOT NULL,
    industry VARCHAR(100) NOT NULL,
    description TEXT NOT NULL,
    requirements TEXT NOT NULL,
    status ENUM('active', 'expired') NOT NULL DEFAULT 'active',
    publish_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (company_id) REFERENCES company(id) ON DELETE CASCADE
);
```

### 4.6 创建简历表

```sql
CREATE TABLE IF NOT EXISTS resume (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    student_id BIGINT NOT NULL,
    resume_name VARCHAR(100) NOT NULL,
    basic_info JSON NOT NULL,
    education JSON NOT NULL,
    experience JSON,
    projects JSON,
    skills JSON,
    status ENUM('draft', 'published') NOT NULL DEFAULT 'draft',
    create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (student_id) REFERENCES student(id) ON DELETE CASCADE
);
```

### 4.7 创建投递记录表

```sql
CREATE TABLE IF NOT EXISTS application (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    student_id BIGINT NOT NULL,
    job_id BIGINT NOT NULL,
    resume_id BIGINT NOT NULL,
    status ENUM('pending', 'reviewed', 'interview', 'offered', 'rejected') NOT NULL DEFAULT 'pending',
    create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (student_id) REFERENCES student(id) ON DELETE CASCADE,
    FOREIGN KEY (job_id) REFERENCES job(id) ON DELETE CASCADE,
    FOREIGN KEY (resume_id) REFERENCES resume(id) ON DELETE CASCADE
);
```

### 4.8 创建评价表

```sql
CREATE TABLE IF NOT EXISTS evaluation (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    evaluator_id BIGINT NOT NULL,
    evaluatee_id BIGINT NOT NULL,
    type ENUM('company', 'student') NOT NULL,
    score INT NOT NULL CHECK (score BETWEEN 1 AND 5),
    content TEXT NOT NULL,
    create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP
);
```

### 4.9 创建岗位收藏表

```sql
CREATE TABLE IF NOT EXISTS job_collection (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    student_id BIGINT NOT NULL,
    job_id BIGINT NOT NULL,
    create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    UNIQUE KEY uk_student_job (student_id, job_id),
    FOREIGN KEY (student_id) REFERENCES student(id) ON DELETE CASCADE,
    FOREIGN KEY (job_id) REFERENCES job(id) ON DELETE CASCADE
);
```

### 4.10 创建面试记录表

```sql
CREATE TABLE IF NOT EXISTS interview (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    application_id BIGINT NOT NULL,
    interview_time DATETIME NOT NULL,
    interview_location VARCHAR(255) NOT NULL,
    interview_type VARCHAR(50) NOT NULL,
    interviewer VARCHAR(50) NOT NULL,
    notes TEXT,
    status ENUM('scheduled', 'completed', 'cancelled') NOT NULL DEFAULT 'scheduled',
    create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    update_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (application_id) REFERENCES application(id) ON DELETE CASCADE
);
```

### 4.11 创建系统日志表

```sql
CREATE TABLE IF NOT EXISTS system_log (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT,
    operation VARCHAR(100) NOT NULL,
    description TEXT NOT NULL,
    ip_address VARCHAR(50),
    create_time DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP
);
```

## 5. 索引创建

```sql
-- 用户表索引
CREATE INDEX idx_user_role ON user(role);
CREATE INDEX idx_user_status ON user(status);

-- 学生表索引
CREATE INDEX idx_student_user_id ON student(user_id);
CREATE INDEX idx_student_major ON student(major);
CREATE INDEX idx_student_grade ON student(grade);

-- 企业表索引
CREATE INDEX idx_company_user_id ON company(user_id);
CREATE INDEX idx_company_name ON company(company_name);
CREATE INDEX idx_company_industry ON company(industry);
CREATE INDEX idx_company_status ON company(status);

-- 岗位表索引
CREATE INDEX idx_job_company_id ON job(company_id);
CREATE INDEX idx_job_title ON job(job_title);
CREATE INDEX idx_job_location ON job(location);
CREATE INDEX idx_job_industry ON job(industry);
CREATE INDEX idx_job_status ON job(status);
CREATE INDEX idx_job_publish_time ON job(publish_time);
CREATE INDEX idx_job_salary ON job(salary_min, salary_max);

-- 简历表索引
CREATE INDEX idx_resume_student_id ON resume(student_id);
CREATE INDEX idx_resume_status ON resume(status);

-- 投递记录表索引
CREATE INDEX idx_application_student_id ON application(student_id);
CREATE INDEX idx_application_job_id ON application(job_id);
CREATE INDEX idx_application_status ON application(status);
CREATE INDEX idx_application_create_time ON application(create_time);

-- 评价表索引
CREATE INDEX idx_evaluation_evaluator_id ON evaluation(evaluator_id);
CREATE INDEX idx_evaluation_evaluatee_id ON evaluation(evaluatee_id);
CREATE INDEX idx_evaluation_type ON evaluation(type);

-- 岗位收藏表索引
CREATE INDEX idx_job_collection_student_id ON job_collection(student_id);
CREATE INDEX idx_job_collection_job_id ON job_collection(job_id);

-- 面试记录表索引
CREATE INDEX idx_interview_application_id ON interview(application_id);
CREATE INDEX idx_interview_time ON interview(interview_time);
CREATE INDEX idx_interview_status ON interview(status);
```

## 7. 数据库优化建议

1. **使用连接池**：配置合适的数据库连接池，提高数据库访问效率
2. **分区表**：对于数据量大的表（如投递记录、系统日志），考虑使用分区表
3. **读写分离**：在高并发场景下，考虑实现读写分离
4. **缓存策略**：对于热点数据（如岗位信息），使用Redis缓存
5. **定期维护**：定期进行数据库备份、优化和清理
6. **监控**：建立数据库监控系统，及时发现和解决问题

## 8. 安全考虑

1. **密码加密**：用户密码使用bcrypt等安全算法加密存储
2. **SQL注入防护**：使用参数化查询，避免SQL注入攻击
3. **数据验证**：对所有输入数据进行严格验证
4. **权限控制**：严格的数据库访问权限控制
5. **敏感数据保护**：对敏感数据进行加密存储
6. **审计日志**：记录所有重要操作的审计日志
