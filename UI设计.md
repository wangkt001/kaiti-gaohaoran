# 大学生就业信息管理系统UI设计

## 1. 设计理念

### 1.1 设计风格

- **整体风格**：现代、简洁、专业
- **配色方案**：主色调为蓝色系（#1890ff），辅助色为白色和浅灰色
- **字体**：系统默认字体，标题使用稍大字号和加粗效果
- **布局**：响应式设计，适配PC端和移动端

### 1.2 设计原则

- **用户体验优先**：界面简洁明了，操作流程顺畅
- **视觉层次清晰**：通过颜色、字体大小、间距等元素建立清晰的视觉层次
- **一致性**：保持各页面设计风格的一致性
- **无障碍设计**：考虑不同用户的使用需求

## 3. 响应式设计

### 3.1 PC端（≥1200px）

- 完整布局，左侧导航栏固定
- 内容区充分利用空间
- 多列布局

### 3.2 平板端（768px-1199px）

- 左侧导航栏可折叠
- 内容区适当调整布局
- 保持核心功能可见

### 3.3 移动端（<768px）

- 顶部导航栏
- 底部导航菜单
- 单列布局
- 简化界面元素

## 6. 技术实现

### 6.1 前端框架

- **Vue 3**：核心框架
- **Element Plus**：UI组件库
- **ECharts**：数据可视化
- **Vue Router**：路由管理
- **Pinia**：状态管理

### 6.2 响应式实现

- **CSS Grid**：网格布局
- **Flexbox**：弹性布局
- **媒体查询**：适配不同屏幕尺寸
- **rem/em**：相对单位

### 6.3 性能优化

- **组件懒加载**：按需加载组件
- **图片优化**：压缩图片，使用合适格式
- **代码分割**：减小初始加载体积
- **缓存策略**：合理使用浏览器缓存

## 7. 设计规范

### 7.1 色彩规范

- **主色调**：#1890ff（蓝色）
- **辅助色**：#52c41a（绿色）、#faad14（黄色）、#f5222d（红色）
- **中性色**：#f5f5f5（浅灰）、#e8e8e8（中灰）、#8c8c8c（深灰）、#262626（黑色）

### 7.2 字体规范

- **标题**：18px-24px，加粗
- **正文**：14px-16px，常规
- **辅助文字**：12px，常规
- **行高**：1.5-1.8

### 7.3 间距规范

- **大间距**：24px-32px
- **中间距**：16px-20px
- **小间距**：8px-12px
- **微小间距**：4px-6px

### 7.4 圆角规范

- **按钮**：4px-8px
- **卡片**：8px-12px
- **输入框**：4px-6px

### 7.5 阴影规范

- **轻微阴影**：0 2px 8px rgba(0, 0, 0, 0.09)
- **中等阴影**：0 4px 16px rgba(0, 0, 0, 0.12)
- **强烈阴影**：0 8px 32px rgba(0, 0, 0, 0.15)

## 8. 页面流程图

### 8.1 学生求职流程

```
登录 → 浏览岗位 → 搜索/筛选岗位 → 查看岗位详情 → 投递简历 → 查看投递状态 → 面试/录用
```

### 8.2 企业招聘流程

```
登录 → 发布岗位 → 接收简历 → 筛选简历 → 安排面试 → 录用管理 → 评价学生
```

### 8.3 管理员管理流程

```
登录 → 用户管理 → 企业审核 → 数据统计 → 系统设置
```
